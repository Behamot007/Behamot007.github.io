# Service: Arena

## Zweck
Dieser Ordner dient als Platzhalter für zukünftige Arena-Analysen (z. B. Match- oder Stat-Übersichten). Hier werden eigenständige Frontend-Seiten für den Arena-Stack abgelegt, sobald sie verfügbar sind.
